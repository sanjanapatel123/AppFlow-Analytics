# 🚀 AppFlow Analytics

A modern SaaS Product Analytics Dashboard built with Next.js, React, TypeScript, Tailwind CSS, Zustand, React Query, and React Flow.

Inspired by products like AtlasGrid, Linear, and Vercel Dashboard.

---

## ✨ Features

### Authentication
- Login
- Form Validation (React Hook Form + Zod)
- Protected Routes
- Session Restore
- Remember Me
- Logout
- Zustand Authentication Store

### Dashboard
- KPI Cards
- Responsive Layout
- Dark / Light Mode
- Sidebar Navigation
- Loading Skeletons
- Empty State
- Error State

### Analytics
- Interactive Charts
- Weekly User Activity
- Top Screen Analytics
- Responsive Charts

### Screen Management
- Searchable Data Table
- Status Badges
- Screen Performance
- Visitor Statistics

### Flow Visualization
- React Flow Graph
- Custom Nodes
- Search Nodes
- Node Details Panel
- Connected Path Highlight
- Interactive Graph

### Architecture
- Next.js App Router
- TypeScript
- Feature Based Folder Structure
- Service Layer
- API Layer
- React Query
- Zustand
- Axios
